Even if the local storage data is closed by the browser, these data are stored for a long time, but the session storage data is cleared immediately when the browser is turned off .

Any variable in the global scope can be accessed from any subsequent scope but any variable in block scope cannot be accessed from any subsequent scope.

Event loop is an endless loop, Which waits for tasks, executes them and then sleeps until it receives more tasks. The event loop executes tasks from the event queue only when the call stack is empty i.e. there is no ongoing task. The event loop allows us to use callbacks and promises.
